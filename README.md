# 🌐 FactZone.online

**FactZone.online** एक आधुनिक और रेस्पॉन्सिव वेबसाइट है जो दुनिया भर के रोचक तथ्यों (amazing facts), विज्ञान, इतिहास और रहस्यमयी जानकारियों को पाठकों तक पहुँचाती है।

---

## 🚀 मुख्य फीचर्स (Features)

* **कैटेगरी आधारित कंटेंट:** विज्ञान, इतिहास, अंतरिक्ष और सामान्य ज्ञान के तथ्य।
* **रेस्पॉन्सिव डिज़ाइन:** मोबाइल, टैबलेट और डेस्कटॉप पर बेहतरीन यूजर एक्सपीरियंस।
* **फास्ट लोडिंग:** ऑप्टिमाइज्ड कोड और बेहतरीन परफॉर्मेंस।
* **सोशल शेयरिंग:** फैक्ट्स को सीधे सोशल मीडिया पर शेयर करने की सुविधा।

---

## 🛠️ इस्तेमाल की गई तकनीकें (Tech Stack)

* **Frontend:** HTML5, CSS3, JavaScript / Tailwind CSS
* **Backend:** WordPress / Node.js (अपनी तकनीक के अनुसार बदलें)
* **Hosting:** Hostinger / DigitalOcean / Vercel (अपनी होस्टिंग के अनुसार बदलें)

---

## 💻 लोकल सिस्टम पर कैसे चलाएं (Installation)

परियोजना को अपने लोकल कंप्यूटर पर सेटअप करने के लिए नीचे दिए गए स्टेप्स का पालन करें:

1. **रिपॉजिटरी क्लोन करें:**
   ```bash
   git clone https://github.com
   ```

2. **प्रोजेक्ट फोल्डर में जाएं:**
   ```bash
   cd factzone.online
   ```

3. **फाइलें ओपन करें:**
   `index.html` फाइल को किसी भी ब्राउज़र में खोलें या लोकल सर्वर (जैसे Live Server) का उपयोग करें।

## 🤝 योगदान कैसे दें (Contributing)

यदि आप इस प्रोजेक्ट को बेहतर बनाने में मदद करना चाहते हैं:
1. इस रिपॉजिटरी को **Fork** करें।
2. अपनी नई फीचर के लिए एक **Branch** बनाएं (`git checkout -b feature/AmazingFeature`)।
3. अपने बदलावों को **Commit** करें (`git commit -m 'Add some AmazingFeature'`)।
4. ब्रांच में **Push** करें (`git push origin feature/AmazingFeature`)।
5. एक **Pull Request** ओपन करें।
